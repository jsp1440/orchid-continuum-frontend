# UI-0001 — Homepage Readability Standard

Status: Active guidance
Date: 2026-06-10

## Purpose

The Orchid Continuum homepage must be readable for orchid society members, researchers, partners, donors, students, and visitors using phones or tablets. Visual elegance cannot come at the expense of legibility.

## Core rule

If a visitor cannot comfortably read a caption, story, badge, or section description, the design has failed no matter how beautiful the layout is.

## Minimum readability requirements

### Text size

- Body text: minimum 18px on desktop/tablet.
- Story captions: 18–20px preferred.
- Supporting labels and metadata: never below 12px unless purely decorative.
- Critical labels, captions, and calls to action should be 14px or larger.

### Contrast

Avoid critical text combinations such as:

- light gray on ivory
- pale gold on ivory
- low-opacity text on textured backgrounds
- thin script over images
- small serif text over gradients

Required combinations:

- dark background + bright readable text, or
- light background + dark readable text.

Critical content should meet WCAG AA contrast expectations.

### Font weight

- Body text should use medium weight where possible.
- Avoid ultra-light or hairline fonts for captions.
- Headings may use elegant serif fonts, but not too thin.

### Captions

Caption boxes below galleries or image grids must use:

- high-contrast foreground/background
- 18px readable text where space allows
- line-height around 1.6–1.8
- no low-opacity text for essential content
- maximum line length that does not strain reading

### Knowledge badges

Knowledge badges should be visible, not decorative.

- Badge numbers must be readable at a glance.
- Icons should have sufficient contrast.
- Zero-count badges may be muted, but nonzero badges should be visually clear.

### Mobile and tablet

The homepage should be checked on:

- iPad/tablet width
- phone width
- outdoor-brightness conditions if possible
- age-50+ readability expectations

## Genus-of-the-Day rule

The Genus-of-the-Day carousel is a museum exhibit, not a fine-print caption. Species names, captions, stories, and knowledge badges should be readable without zooming.

## Design principle

Beauty supports comprehension. It must not obscure it.
